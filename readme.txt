== Description ==
RTV Disable Plugin Updates is a custom WordPress plugin that allows you to manage and disable updates for specific plugins.

== Features ==
- Disable updates for selected plugins directly from a settings page.
- View and select from all installed plugins (active or inactive).
- Export and import ignore lists in JSON format.
- Reset ignore lists to defaults.
- Advanced settings to ignore specific versions of plugins.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/rtv-disable-plugin-updates` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Plugins > Ignored Plugins to manage the list of plugins to ignore.
